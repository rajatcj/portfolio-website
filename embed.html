Link Preview API

This API allows you to create a link preview by setting the values of the following meta tags:

  • og:title
  • og:site_name
  • og:description
  • og:image
  • theme-color
  • twitter:card

To use the API, simply link to this page with the desired values as query parameters in the URL. For example, to create a link preview with a title of "Welcome to Cookie and Rock" and a site name of "Cookie and Rock", you can use the following URL: 

https://example.com/link-preview.html?title=Welcome%20to%20Cookie%20and%20Rock&site_name=Cookie%20and%20Rock
